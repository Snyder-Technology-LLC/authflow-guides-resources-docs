## üöÄ Quick Start

Get your paywall live in **10‚Äì15 minutes**:

1. **Create an Authflow account** ‚Üí [authflow.ai](https://authflow.ai)
2. **Connect Stripe** ‚Üí Follow the onboarding wizard
3. **Create a paywall** ‚Üí Set your price, trial period, and billing cycle
4. **Copy the GPT schema + instructions** ‚Üí From your paywall details page
5. **Paste into your custom GPT** ‚Üí In the Actions configuration
6. **Publish and start earning** ‚Üí Share your GPT link

> [!TIP]
> **If you can copy and paste, you can implement a paywall.** No coding required ‚Äî just simple configuration.

### Integration Checklist

- [ ] Paywall created in Authflow dashboard
- [ ] OpenAPI schema pasted into GPT Actions
- [ ] API key configured (Authentication ‚Üí API Key ‚Üí Custom Header)
- [ ] Privacy policy URL added
- [ ] Authflow instructions added at the **top** of your GPT instructions
- [ ] Tested with both subscribed and unsubscribed users

> [!IMPORTANT]
> Always place Authflow's context instructions at the **very top** of your GPT's instruction field. This ensures the paywall logic runs before any other behavior.

## ‚ùì Frequently Asked Questions

<details>
<summary><strong>Is coding required?</strong></summary>

<br>

No! Authflow is built as a **low-code/no-code solution**. You'll interact through a simple dashboard where you can create paywalls, choose subscription models, and copy/paste the integration snippets. If you can edit a GPT's instructions and add an action, you can use Authflow.

</details>

<details>
<summary><strong>How do I get paid?</strong></summary>

<br>

All payouts flow directly through your connected **Stripe account**. You can choose daily, weekly, or monthly payouts into your bank account. Stripe provides full transparency into transactions, chargebacks, and payout schedules.

</details>

<details>
<summary><strong>What are the fees?</strong></summary>

<br>

Authflow charges a **flat 10% fee** on all successful transactions. There are no monthly subscription costs or hidden fees. For enterprise users processing high volume, custom pricing can be negotiated. See [Authflow Pricing Plans](https://github.com/Snyder-Technology-LLC/authflow-guides-resources-docs/blob/main/content/paywalls/authflow-pricing-plans.md) for details.

</details>

<details>
<summary><strong>Is it secure?</strong></summary>

<br>

Absolutely. Authflow never stores sensitive card information ‚Äî we rely entirely on **Stripe**, which is PCI DSS Level 1 compliant (the highest security certification for payments). All traffic is fully encrypted, and authentication is handled through secure tokens and API keys.

</details>

<details>
<summary><strong>Can I use one paywall for multiple GPTs?</strong></summary>

<br>

Yes! You can create multiple paywalls to monetize different GPTs or offer varied pricing tiers. Copy the same schema into multiple GPTs ‚Äî users authenticate once and can access all GPTs sharing that paywall.

</details>

<details>
<summary><strong>Can I offer free trials or discounts?</strong></summary>

<br>

Yes! Authflow lets you configure **free trial periods** for subscriptions, so users can try your GPT before committing. You can also create discount codes and promotional pricing to increase conversions.

</details>

<details>
<summary><strong>What if I hit the 8,000 character limit?</strong></summary>

<br>

See our [Character Limit Troubleshooting](https://github.com/Snyder-Technology-LLC/authflow-guides-resources-docs/blob/main/content/troubleshooting/character-limit.md) guide. Tips: prioritize Authflow instructions at the top, optimize your custom instructions, and use the Knowledge Base for long content.

</details>
