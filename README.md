## What is Authflow?

Authflow is the first ChatGPT-native paywall system for monetizing custom GPTs. It handles:

- **Authentication** ‚Äî Verify subscribers via email
- **Payments** ‚Äî Powered by Stripe (secure, global)
- **Subscriptions** ‚Äî Recurring billing, free trials, one-time payments
- **Analytics** ‚Äî Track revenue, subscribers, and conversions

### Who is this for?

- GPT creators who want to earn from their work
- Businesses offering AI-powered tools
- Educators, coaches, and content creators
- Anyone with a valuable custom GPT

## Common Questions

<details>
<summary><strong>Is coding required?</strong></summary>

No traditional coding ‚Äî copy/paste configuration only. If you can edit a GPT's instructions and add an action, you can use Authflow.

</details>

<details>
<summary><strong>How do I get paid?</strong></summary>

Payments go directly to your connected Stripe account. Stripe handles payouts according to your settings (typically weekly or monthly).

</details>

<details>
<summary><strong>What are the fees?</strong></summary>

See the [Authflow Pricing Plans](https://github.com/Snyder-Technology-LLC/authflow-guides-resources-docs/blob/main/content/paywalls/authflow-pricing-plans.md) for current fees. All plans charge a transaction fee plus standard Stripe processing fees.

</details>

<details>
<summary><strong>Can I use one paywall for multiple GPTs?</strong></summary>

Yes! Copy the same schema into multiple GPTs. Users authenticate once and can access all GPTs sharing that paywall.

</details>

<details>
<summary><strong>What if I hit the 8,000 character limit?</strong></summary>

See [Character Limit Troubleshooting](https://github.com/Snyder-Technology-LLC/authflow-guides-resources-docs/blob/main/content/troubleshooting/character-limit.md). Tips: prioritize Authflow instructions at the top, optimize your custom instructions, use the Knowledge Base for long content.

</details>
